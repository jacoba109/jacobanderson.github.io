Projects

HuskyNav – AI Campus Navigation Assistant

Gemini • LangChain • React • Python • Google Maps API

HuskyNav is an AI-powered chatbot-style navigation system for new students on campus. It guides users from their current location all the way to specific rooms inside university buildings, including step-by-step indoor directions.

  • Built a full-stack system with a React frontend and Python backend, integrating Google Maps API for outdoor routing and AI-generated indoor directions.
  • Used Gemini and LangChain to structure navigation prompts, retrieve building/floor information, and generate clear, human-readable directions.

GNN for Molecular Solubility Prediction

Spektral • TensorFlow • Ray Tune • Python

Research assistant project benchmarking a graph neural network that predicts aqueous solubility from molecular structure using the AqSolDB dataset (~10,000 molecules).

  • Prepared the dataset, converted molecular structures into graph representations, and configured the model for regression.
  • Ran large-scale hyperparameter sweeps (500 models) with Ray Tune and achieved strong performance (MSE ≈ 0.25) compared to a naive baseline.

Imagle – Siamese Network Image Matching

TensorFlow • Python • Web App

A Wordle-inspired image comparison game powered by a siamese neural network that learns visual similarity between images.

  • Designed and trained a siamese architecture to embed images into a feature space where similar images are close together.
  • Implemented the game logic and UX to let users iteratively guess the target image based on model feedback.
